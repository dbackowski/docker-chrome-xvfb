# docker-chrome-xvfb

Dockerfile for CI (continuous integration) based on official node docker image with xvfb, google-chrome for headless testing.

## What is included?

* Node.js - 6.11.1
* xvfb
* Google Chrome - 60.0.3112.78

## Usage 

```
docker run damianbackowski/docker-chrome-xvfb [cmd]
```
